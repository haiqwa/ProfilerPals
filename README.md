# ProfilerPals
提供各类AI 硬件性能分析工具使用的教程和案例。

欢迎来到本仓库！🚀

在本仓库中，你可以：
- 学习常见AI硬件供应商的Profile工具教程
    - Nvidia
    - MUSA
    - MLU
- 直接查看现成的 Demo Trace：我们提供了多个典型场景下的 trace 文件，帮助你快速了解该 profile 工具的能力与输出格式，判断它是否满足你的性能分析需求。
- 运行代码生成自己的 Trace：通过执行仓库中的示例脚本，你可以亲手 dump 出新的 trace 文件，边做边学，掌握 profile 工具的基本使用方法和调试技巧。
- 获取最新使用注意事项：我们会在 README 中持续更新 profile 工具的使用技巧、常见陷阱及避坑指南，助你高效、准确地进行性能剖析。




无论你是初次接触性能分析，还是希望深入理解工具行为，这个仓库都能为你提供实用的参考和快速上手的路径。快来看看有没有你需要的示例吧！🔍
